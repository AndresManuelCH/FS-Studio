The current folder structure is:

- DB: contains the sql files that allow you to create the database (its own creation if it does not exist, 
as well as creating each of the tables that make it up) .

- dist: contains the jar of the project that is created when compiling in NetBeans (where it has been 
developed) as well as the generated README.txt and the lib folder (which contains the external libraries 
used for the correct functioning of the developed module).

- External libraries: contains the external libraries necessary for the correct functioning of the developed 
module.

- nbproject: contains the necessary files that NetBeans creates for its correct opening in said IDE.

- src: contains the corresponding Java files with the classes that the developed module has and 
indicated in the class diagram.

- LICENSE.txt: license file GNUGeneralPublicLicenceV3.
